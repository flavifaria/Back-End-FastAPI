A Jornada da Requisição (FastAPI)

Passe o mouse sobre as caixas para ver os detalhes de cada arquivo.

Postman

Cliente

O Cliente (Front-end)

Ferramenta Externa

Simula um aplicativo ou site. É onde montamos o JSON, os Cabeçalhos (Headers) e enviamos a requisição HTTP (GET, POST).

🍕 Analogia: O cliente faminto fazendo o pedido.
→

Router

Endpoints

Roteador (Endpoints)

routers.py

A porta de entrada da API. Define as URLs (ex: /produtos) e os métodos. Recebe a chamada HTTP e direciona para a função certa.

🍕 Analogia: O Garçom que anota o seu pedido e o número da sua mesa.
→

Auth Guard

Segurança / JWT

Guarda de Autenticação

auth.py / dependencies.py

Intercepta a requisição para extrair o Token JWT do cabeçalho. Verifica se o usuário está logado e tem permissão para continuar.

🍕 Analogia: O Segurança da porta checando se seu nome está na lista de reservas.
→

Schema

Pydantic

Validação de Dados

schemas.py

Usa a biblioteca Pydantic para garantir que o formato do JSON está perfeito. Barra textos em campos de número (gera Erro 422).

🍕 Analogia: O Fiscal conferindo se o pedido faz sentido (ex: barrar pizza de pneu).
→

Controller

Orquestrador

Controlador

controllers.py

O cérebro do fluxo. Recebe os dados validados, aciona os Serviços necessários, junta as respostas e devolve para o Router enviar ao usuário.

🍕 Analogia: O Gerente do salão que organiza quem vai fazer o quê.
→

Service

Regras de Negócio

Regras de Negócio

services.py

Onde a mágica acontece. Faz cálculos, verifica lógicas complexas (ex: "tem saldo?"), dispara e-mails e processa as informações.

🍕 Analogia: O Chef de Cozinha preparando a comida de fato.
→

Repository

SQL / Queries

Comunicação com o Banco

repositories.py

O único lugar que entende de Banco de Dados. Isola os comandos SQL ou ORM (ex: SQLAlchemy). Busca e salva as informações físicas.

🍕 Analogia: O Estoquista que vai até o fundo buscar os ingredientes.
→

Database

SQLite

Banco de Dados

banco.db / database.py

A memória permanente da aplicação. É onde as tabelas vivem e os registros (usuários, produtos) ficam salvos no disco.

🍕 Analogia: O Armazém ou despensa do restaurante.
